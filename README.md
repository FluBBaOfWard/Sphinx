# Sphinx V0.4.1

Bandai WonderSwan(Color/Crystal) System On a Chip emulator for GBA/NDS.

Make sure RAM is aligned to 16 bytes, the sound mixer is dependant on this.
After reset also call wsvSetCartOk.
